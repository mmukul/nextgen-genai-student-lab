# NextGen GenAI Student Lab

A beginner-friendly local AI lab built with:

- Streamlit
- FastAPI
- Ollama
- Llama 3.2

## Installation

```bash
git clone https://github.com/mmukul/nextgen-genai-student-lab.git

cd nextgen-genai-student-lab

chmod +x *.sh

./install.sh
```

## Start

```bash
./start.sh
```

## Stop

```bash
./stop.sh
```

## Update

```bash
./update.sh
```

## Open

http://localhost:8501

- AI Chat
